工作中Mac使用及技巧

2017-11-11 工作记录 工作记录 Comments Word Count: 351(words) Read Count: 1(minutes)

一.使用软件

1.intellj idea

java开发必备工具，当然也有eclipse可以选择。看习惯。

2.数据库工具

无。。求推荐一个好用稳定的工具。

使用过能找到的基本上都不太顺手，崩溃的崩溃，乱码的乱码。干脆就直接命令行走起了。其实影响也不大。。

(以前写的。。Sequel Pro不错)

3.kerberos配置（可选）

服务器需要身份票据认证，配置客户端

二.基本配置

短命令

有时候打开软件要输入长的命令，可以修改用户目录~/.bashrc的配置

在文件最后添加下面的命令

格式：alias 你的别名=”对应的命令”

这样在命令行直接输入别名就相当于执行这条命令了

例如：

alias idea=”nohup /data/software/idea-IU-171.4424.56/bin/idea.sh &”

alias 57=”ssh root@10.12.12.12“

alias navicat=”nohup /data/software/navicat112_mysql_cs_x64/start_navicat &”

alias mysql=”mysql -u root -p”

alias 169=”ssh user@192.168.100.169“

dns/host

这就不多解释了

vim /etc/resolv.conf

﻿vim /etc/hosts

三.常用命令

1.查询某个文件从第1行到10行的内容

sed -n ‘1,10p’ /data/log/1.txt

2.在某个文件中搜索关键字

grep -iE “error|Exception” /data/log/1.txt

-i 忽略大小写

-E 支持正则表达式

3.清空文件内容

>filename
